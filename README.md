# miniproject_hanhae66
### 항해66 블라인드 게시판
  - 항해가 66일 남은 날 프로젝트를 시작하였고, 항해의 뒷 모습 이란 생각에 항해 66으로 이름을 지었다.
  - 항해66은 항해를 하는 사람들의 익명 게시판이다. Blind를 벤치 마킹 하였다. 
  - 모든 유저는 아이디의 첫 글자, 주특기와 기수만 공개된다.  ( ex) 14th Spring u*****)
